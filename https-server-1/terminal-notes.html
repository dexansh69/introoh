Navigation & File Management

pwd Print current directory path
cd foldername Navigate into a folder
cd .. Go back one directory
cd ~ Go to home directory
ls List files and folders in current directory
mkdir foldername Create a new directory/folder
touch filename Create a new empty file
mv old new Move files/folders — also used to rename files
cp file dest Copy a file to destination
cp -r dir dest Copy a directory (must use -r for folders)
rm filename Remove/delete a file
rm -r foldername Remove/delete a folder
🔍

ls Flags & Listing

ls -l Detailed list with permissions, size, date (A→Z)
ls -lr Detailed list in reverse order (Z→A)
ls -la Show hidden files too (like .zshrc)
ls -lt Sort by newest modified first
ls -ltr Sort by oldest modified first
ls -lS Sort by largest file size first
ls -lSr Sort by smallest file size first
ls -lR List recursively through all subfolders
ls *.js List only .js files (* = wildcard/anything)
ls foldername Peek inside a folder without navigating into it
Small -r = reverse order | Capital -R = recursive (search all subfolders)
📄

Viewing File Contents

// cat · vi · echo
cat filename Print entire file contents to terminal
vi filename Open file in vi editor
i → esc → :wq! vi: press i to insert, esc to exit insert, :wq! to save & quit (remove w to quit without saving)
echo 'hello' Display text in terminal
echo 'text' >> file Append text to a file without overwriting
// head & tail
head file Show first 10 lines of file (default)
head -20 file Show first 20 lines of file
tail file Show last 10 lines of file (default)
tail -20 file Show last 20 lines of file
tail -n +25 file Show everything from line 25 to end
tail -f file Live follow file updates — great for log files!
tail -n +25 file | head -n 5 Show lines 25 to 29 (start at 25, take 5)
🔐

Permissions (chmod)

// permission string breakdown
rw- r-- r--

rw- = user/owner (read ✅ write ✅ execute ❌)  |  r-- = group (read only)  |  r-- = others (read only)

// who is u, g, o, a?
LetterMeansWho
uuserfile owner
ggroupusers in same group
ootherseveryone else
aalleveryone (u+g+o)
// permission letters
LetterPermissionNumber
rread4
wwrite2
xexecute1
// chmod commands
chmod u+r file Give owner read permission
chmod o-w file Remove write from others
chmod a+x file Give everyone execute permission
chmod g+rwx file Give group all permissions
chmod 664 file Numbers: u=6(rw) g=6(rw) o=4(r) — syntax: chmod [u][g][o] filename
🔎

grep — Search Inside Files

grep "text" file Search for text in a file
grep -i "text" file Case insensitive search
grep -n "text" file Show line numbers of matches
grep -c "text" file Count how many lines match
grep -v "text" file Show lines that DON'T match
grep -w "text" file Whole word match only
grep -r "text" . Search recursively in all subfolders
grep -l "text" *.txt Show only filenames that match
grep -A 2 "text" file Show 2 lines after match
grep -B 2 "text" file Show 2 lines before match
grep -o "text" file Show only the matched part, not whole line
ls -lR | grep .json Pipe ls output into grep to find all .json files
grep "text" file | wc -l Count how many lines contain the text
✏️

sed — Find & Replace in Files

s = substitute | d = delete | p = print | -n = suppress default output | -i = edit file directly
sed 's/old/new/' file Replace first occurrence per line
sed 's/old/new/g' file Replace all occurrences
sed -i 's/old/new/g' file Edit the file directly (Mac: use -i '')
sed -ibackup 's/old/new/' file Edit file and create backup (file.backup)
sed '/ERROR/d' file Delete lines containing ERROR
sed -n '/ERROR/p' file Print only lines containing ERROR
sed '3 s/old/new/' file Replace only on line 3
sed '3,5 s/old/new/' file Replace only on lines 3 to 5
sed -n '3,/ERROR/p' file Print from line 3 until ERROR is found
sed '/pattern/d' file Delete all lines matching pattern
⚙️

awk — Process & Transform Text

awk '{print}' file Print entire file (like cat)
awk '{gsub(/old/,"new")}{print}' file Find and replace all occurrences
awk 'BEGIN{print "HEADER"} {print} END{print "FOOTER"}' file

BEGIN = runs before file | {print} = every line | END = runs after file. Used to add headers/footers to output.

🔢

wc — Word Count

wc file Shows lines, words, characters all at once
wc -l file Count lines only
wc -w file Count words only
wc -c file Count characters only
grep "ERROR" file | wc -l Count how many ERROR lines exist in file
🟢

Node.js Commands

node -v Check if Node is installed + version
npm -v Check npm version
node index.js Run a JavaScript file
npm init Create a new package.json
npm install express Install a package (e.g. express)
npm install Install all dependencies from package.json
app.use(express.json()) — built-in body parser, replaces need for body-parser package
⌨️

Useful Shortcuts

clear Clear the terminal screen
Cmd + K Clear terminal completely (Mac)
Ctrl + L Clear terminal (history still accessible)
Ctrl + C Stop/cancel current running command
Tab Autocomplete folder/file names
↑ arrow key Cycle through previous commands
date +%s Generate current Unix timestamp
